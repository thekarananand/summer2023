# Docker 

### Volume Mapping

``` bash
docker run -v <H>:<C> <IMAGE> 
```

here,
- `<C>` is the dir on container to be mapper
- `<H>` is the dir on host, where the container's dir will be mapper to.

### Login as Other User

``` bash
docker run -u  <USER> <IMAGE> 
```
- `<C>` is the dir on container to be mapper
- `<H>` is the dir on host, where the container's dir will be mapper to.
